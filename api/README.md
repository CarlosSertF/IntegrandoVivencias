# Integrando-Viv-ncias
Projeto para a matéria de POO2

Este projeto consiste em uma aplicação de um sistema que manuseia e armazena informações de pacientes de uma determinada clínica pedagógica da região, que por meio de uma breve anamnese, a psicopedagoga consiga ter uma visão geral sobre o paciente e assim consiga prosseguir com a consulta efetivamente.
